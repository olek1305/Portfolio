# Abex Guilds

<br />

- **Description:** The project is only for the game AFK Arena. You can manage not only one guild,
  but also multiple guilds, add people to the list with injury records on abex from
  multiple seasons, notebooks, transfers to another guild. Can check stats (count
  users, notes, kick, ban, Guild dmg totals depend on the season) Activity is like
  class register, you mark these players whether they have farmed on the day or
  completed the farm, so that you do not leave out these players.
- **Link:** [GitHub](https://github.com/olek1305/AbexGuilds)
- **Language:** PHP, JavaScript
- **Framework:** Laravel, VueJs, Tailwind, Inertia.js
- **end data:** 04.2024
